## Description
The Book Searcher was created to showcase a search database with your favorite books in mind.

This application was build with implementing graphql.  It has and updated backend and front end using mutations and schemas.

## User Story

```md
AS AN avid reader
I WANT to search for new books to read
SO THAT I can keep a list of books to purchase
```

## Technologies Used

The Book Searcher integrates various technologies:

**Front-End Development:**

- Style: CSS
- Script: JavaScript
- Framework: React.js

**Back-End Development:**

- Environment: Node.js
- Server: Express.js
- Database: MongoDB
- API: GraphQL with Apollo Server
- Authentication: JWT
